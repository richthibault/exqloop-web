# Exquisite Loop Web — Project Guide

## Tech Stack
- **Backend:** Java 17, Spring Boot, Thymeleaf (server-side templates)
- **Frontend:** Tailwind CSS (CDN), vanilla JS, Google Fonts
- **Build:** Maven

## Design Context

### Brand
Exquisite Loop LLC — boutique bespoke software development (web & mobile) for businesses.

### Personality
Crafted, thoughtful, precise. A skilled artisan's studio, not a generic agency.

### Visual Direction
- **Dark-mode.** Backgrounds: `#0c0f14` / `#0e1218` / `#080a0e`
- **Accent color:** `#19B5FE` (sky blue) — used sparingly and precisely
- **Typography:** Playfair Display (serif, headlines, italic emphasis) + Inter (body/UI)
- **Photography:** fibonacci shells, moody architecture, laptop workbench — large, desaturated, dark-overlaid
- **No:** purple gradients, generic SaaS templates, bouncy animations

### Design Principles
1. Restraint over decoration — whitespace is a feature
2. Photography over illustration — use images large and confidently
3. One accent (`#19B5FE`), used precisely — its value comes from scarcity
4. Serif for soul, sans for clarity
5. Interactions feel weighty, not playful — slow ease transitions, glow on CTAs
