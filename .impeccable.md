# Exquisite Loop — Design System

## Design Context

### Users
B2B decision-makers (CTOs, project managers, business owners) evaluating a boutique software development partner. They arrive skeptical and need to feel confident they're dealing with a serious, skilled, trustworthy firm — not a generic agency.

### Brand Personality
**Crafted & thoughtful.** Exquisite Loop is a boutique practice, not a body shop. The brand should feel like an artisan's studio: deliberate, precise, and quietly confident.

Three words: **Precise. Intentional. Refined.**

### Aesthetic Direction
- **Dark-mode first.** Near-black backgrounds (`#0c0f14`, `#0e1218`, `#080a0e`) with `#19B5FE` (sky blue) as the sole accent color. White for primary text, mid-grays for secondary.
- **Serif + sans pairing.** `Playfair Display` for headlines and emphasis (italic for key phrases); `Inter` for all body/UI text.
- **Photography-driven.** Fibonacci shell imagery (`shell1.jpg`, `shell2.jpg`), moody architectural shots (`wall.jpg`, `jug.jpg`), and the laptop workbench (`workbench.jpg`) anchor the visual story. Dark-overlaid, desaturated, used at large scale.
- **Grid overlay aesthetic.** Subtle `#19B5FE` grid patterns reinforce the "precision / engineering" idea.
- **Anti-references:** Generic SaaS purple gradients, stock-photo-heavy hero sections, flat "startup template" look, excessive animation.

### Typography
- **Display:** Playfair Display, weight 500, `letter-spacing: -0.01em`, italic for hero emphasis
- **Eyebrow labels:** Inter 600, `font-size: 0.7rem`, `letter-spacing: 0.14em`, uppercase, color `#19B5FE`
- **Body:** Inter 400, `font-size: 0.9375rem`, `line-height: 1.8`, color `#6b7280`

### Design Principles
1. **Restraint over decoration.** Every element should earn its place. Whitespace is a feature, not a gap to fill.
2. **Photography tells the story.** The fibonacci shells, moody architecture, and workbench imagery carry more brand weight than illustrations or icons ever could. Use them large and confidently.
3. **One accent, used precisely.** `#19B5FE` is the only accent. Never dilute it with competing colors. Its value comes from scarcity.
4. **Serif for soul, sans for clarity.** Playfair Display in headlines signals craftsmanship; Inter in body and UI keeps it sharp and legible.
5. **Interactions should feel weighty, not playful.** Transitions 0.2–0.4s ease. Hover lifts (`translateY(-4px)`). Glow effects on CTAs. Nothing bouncy or cartoon-like.
